This plugin protects your registered user base by preventing registration with a disposable email addresse (like mailinator).

== Description ==

This plugin prevent people from registering with a disposable email addresses like the ones provided by mailinator. It protects your most important asset, your registered user base, by preventing contamination by fake accounts. This plugin working principle is similar to spam blacklists.


== Installation ==

1. Unpack the *.zip file and extract the /bdea/ folder and the files.
2. Using an FTP program, upload the full /bdea/ folder to your WordPress plugins directory (Example: /wp-content/plugins).
3. Register at www.block-disposable-email.com and pick your api key.
4. Go to Plugins, click Settings and insert your api key.
5. Go to Plugins and activate the plugin.
